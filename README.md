### Prerequisitos
* [MongoDB](https://www.mongodb.org/)
* [NodeJs](https://nodejs.org/en/)
* [Git](https://git-scm.com/)

##### Node Packages
* npm install -g gulp
* npm install -g bower

## Usage-dev
1. Clonar repositorio
2. Instalar paquetes & dependencias -
`npm install`
`bower install`
3. Inicializar el Servidor con Gulp (2 ventanas)-
`gulp server`
`gulp watchFront`
4. `http://localhost:4000/`
